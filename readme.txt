Para iniciar o programa, você deve apenas rodar o comando "make" no terminal, que ele irá automaticamente
compilar o programa e criar o arquivo "exe".

Basta rodar o arquivo exe (./exe) que irá rodar o código. O uso do sistema é intuitivo.

Para gerar as bibliotecas solicitadas, basta rodar o comando "make lib", que irá compilar automaticamente
tanto a estática quanto a dinâmica.
